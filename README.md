## Install
```shell
yarn install
```

## Config

```shell
yarn Setting 
```

## Login

```shell
yarn masuk 
```

## Profile
untuk melihat akun dan balance shopee pay serta untuk ganti alamat 
```shell
yarn profile 
```


## Get Detail Produk
````shell
yarn detail
````
jika ingin menyesuaikan data silahkan edit di db.json


## Run Test Order
```shell
yarn start order
```

## Run FS
```shell
yarn start
```
